## 프로젝트명

- 바로나 마취통증의학과 디자인 리뉴얼 & 코드 리팩토링

## 리뉴얼 계획

- 디자인 변경
- 모바일 웹 -> 반응협 웹 변경
- 코드 리팩토링

## 개발환경

- **Client Side**: HTML5, CSS3, JS, Jquery
- **Server Side**: JSP
- **Framework**: Spring, Mybatis, Tiles
- **Database**: Mysql
- **Server**: Tomcat8
- **Build**: Maven
